通过学习前置课程，对Python开发、测试，深度学习以及transformer架构有了具体且深入的了解，从今天开始学习大模型相关内容。

然而“大模型”是一个庞大且复杂的类别，想要完善、体系地学习，需要整理清晰的脉络。

我将通过GitHub上热门的LLM学习开源项目以及一些热门博客，进行对LLM技术栈梳理与学习。

- [happy-llm: 📚 从零开始构建大模型](https://github.com/datawhalechina/happy-llm)

  更偏基础，有详细的基础教学、训练过程，对于LLM的应用进行了简单介绍。正如README所言，该项目能够帮助读者**深入了解大语言模型的原理和训练过程**。

- [so-large-lm: 大模型基础: 一文了解大模型基础知识](https://github.com/datawhalechina/so-large-lm)

  同样偏向基础教学，包含了视频课程内容。

- [self-llm: 《开源大模型食用指南》针对中国宝宝量身打造的基于Linux环境快速微调（全参数/Lora）、部署国内外开源大模型（LLM）/多模态大模型（MLLM）教程](https://github.com/datawhalechina/self-llm)

  偏向大模型的环境搭建、部署、微调等实际应用。

- [llm-action:分享大模型相关技术原理以及实战经验（大模型工程化、大模型应用落地）](https://github.com/liguodongiot/llm-action#)

  偏向LLM应用全过程的实践，包含训练、推理、压缩、微调、评测等全过程实践。

- [dive-into-llms: 《动手学大模型Dive into LLMs》系列编程实践教程](https://github.com/Lordog/dive-into-llms)

  偏向LLM实践中关于“安全”相关技术，如水印、隐写、智能体安全等。

- [大模型开发学习专区-昇腾社区](https://www.hiascend.com/edu/growth/lm-development#classification-floor-1)

  在《动手学大模型》原系列教程的基础上，联合华为开发的《大模型开发全流程》系列课程。内容全面，学完就可以入职了。

本项目将结合以上开源项目，按序学习。
